# LLM-Assisted Customer Churn Prediction & Insight System

## Overview
An end-to-end data science and generative AI solution that predicts customer churn and generates natural-language business insights using LLMs.

## Features
- Classical ML churn prediction (Logistic Regression)
- SHAP-based explainability
- Google Gemini powered insight generation
- FastAPI backend

## Tech Stack
Python, scikit-learn, SHAP, Google Gemini API (for natural language insights), FastAPI

## Business Impact
- Enables proactive churn mitigation
- Translates ML outputs into executive-friendly language
